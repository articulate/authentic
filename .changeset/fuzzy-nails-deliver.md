Allows a generic type, which must extend `JWT`, to be defined on Authentic's signature for automatic type casting.

Bumps [jose](https://github.com/panva/jose) to [V5](https://github.com/panva/jose/releases/tag/v5.0.0), this includes a breaking change that only LTS Node versions will be supported by `@articulate/authentic`
